# Manager#lastKey
一番最後から数えて引数で渡された数の位置にある要素を返します。  
  
**Return**: *K*

#### 例を見てください
```js
const manager = new Manager([  
		['key', 'value'],  
		['otherKey', 'otherValue'],  
		['warosuKey', 'warosuValue'],  
		['powerfulKey', 'powerfulValue']  
	]);  
console.log(manager.lastKey()); //output 'powerfulKey'  
```
