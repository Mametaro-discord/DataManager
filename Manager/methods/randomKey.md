# Manager#randomKey
要素をランダムに選んでそのキーを返します。  
  
**Return**: *K*

#### 例を見てください
```js  
const manager = new Manager([  
		['key', 'value'],  
		['otherKey', 'otherValue']  
	]);  
console.log(manager.randomKey()); //output 'key' or 'otherKey'  
```
