# Manager#random
要素をランダムに選んでその値を返します。  
  
**Return**: *V*

#### 例を見てください
```js  
const manager = new Manager([  
		['key', 'value'],  
		['otherKey', 'otherValue']  
	]);  
console.log(manager.random()); //output 'value' or 'otherValue'  
```
